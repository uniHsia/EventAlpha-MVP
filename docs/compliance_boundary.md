# 合规边界

EventAlpha-MVP 是事件研究和市场分析辅助系统，不是荐股系统，不做自动交易，不构成投资建议。

## 系统可以输出

- 可能受益；
- 可能承压；
- 关注方向；
- 风险因素；
- 验证指标；
- 市场观察信号。

## 系统不输出

- 买入；
- 卖出；
- 目标价；
- 保证收益；
- 杠杆建议；
- 个股确定性结论；
- 自动交易指令。

> 本内容仅用于事件研究和市场分析，不构成投资建议。市场价格可能已提前反映相关信息，投资决策需结合个人风险承受能力。

## 第一版实现要求

- Agent 输出不能包含买入、卖出、目标价或保证收益。
- 资产映射只能表达行业、指数、ETF 或主题方向。
- 复盘只验证此前判断和因果链，不给新的交易指令。
